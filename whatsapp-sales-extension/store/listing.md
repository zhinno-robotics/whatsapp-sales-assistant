## 中文描述

### 标题（最多 55 字符）

AI Sales Copilot — 智能翻译与回复

### 简短描述（最多 132 字符）

WhatsApp Web 上的 AI 销售助手：实时翻译、5 种回复建议、自定义话术生成，助你高效沟通外贸客户。

### 详细描述

**🤖 AI 驱动的 WhatsApp 销售沟通工具**

AI Sales Copilot是一款专为外贸从业者打造的浏览器扩展，深度集成于 WhatsApp Web 侧边栏，不依赖任何第三方服务器，所有数据均存储在你本地浏览器中。

**✨ 核心功能**

📝 **一键翻译**

- 客户消息自动翻译为中文，消除语言障碍
- 输入中文，AI 自动转为地道商务英语再发送
- 支持 DeepSeek / OpenAI / OpenRouter 等多种 AI 服务商

💡 **5 种 AI 回复建议**

- 每条客户消息自动生成 5 种不同风格的英文回复：
  · 专业严谨 — 展现行业专业度
  · 亲切友好 — 拉近客户关系  
  · 促进行动 — 推动客户下单
  · 详细周全 — 提供完整信息
  · 简洁精炼 — 快速高效回复
- 每条建议同时展示中英文，一眼看懂

✍️ **自定义 AI 回复**

- 用中文描述你想表达的内容，AI 自动生成地道英文回复
- 例如输入"告诉客户价格可以打 9 折，本周下单有效"，AI 会生成专业的英文商务邮件

🎨 **优雅体验**

- WhatsApp 移动端风格的侧边栏设计
- 深色/浅色主题自由切换
- 联系人搜索、群组过滤
- 输入框自动扩高

🔒 **隐私安全**

- 所有数据存储在你的浏览器本地（chrome.storage.local）
- 不上传任何聊天记录到第三方服务器
- AI 调用仅使用你配置的 API Key，直接请求你选择的 AI 服务商
- 开源透明，代码可审查

**📋 使用步骤**

1. 安装扩展后，打开 WhatsApp Web (web.whatsapp.com)
2. 点击浏览器工具栏的扩展图标 → "Open Side Panel"
3. 在弹窗设置中填入你的 AI API Key（支持 DeepSeek、OpenAI、OpenRouter）
4. 开始使用 AI 翻译和建议功能

**🆓 完全免费**
本版本为免费版，包含上述所有核心功能。无需许可证密钥，无需付费订阅。

**🔧 支持的 AI 服务商**

- DeepSeek（默认推荐，性价比最高）
- OpenAI（GPT-4o / GPT-4）
- OpenRouter（接入各种模型）

## English Description

### Title

AI Sales Copilot — AI Translator & Smart Replies

### Short Description

AI-powered sales copilot for WhatsApp Web: real-time translation, 5 reply suggestions, custom AI copywriting for efficient B2B communication.

### Detailed Description

**🤖 AI-Powered Sales Communication Tool for WhatsApp Web**

AI Sales Copilot is a browser extension purpose-built for B2B sales and international trade professionals. It deeply integrates into the WhatsApp Web side panel, requires no third-party servers, and stores all data locally in your browser.

**✨ Core Features**

📝 **One-Click Translation**

- Auto-translate customer messages to your native language
- Compose in your native language, send as polished business English
- Supports DeepSeek / OpenAI / OpenRouter AI providers

💡 **5 AI Reply Suggestions**

- Each incoming message generates 5 reply options in distinct tones:
  · Professional — Demonstrate industry expertise
  · Friendly — Build warm customer rapport
  · Closing — Drive toward concrete next steps
  · Detailed — Provide thorough, informative responses
  · Concise — Quick, direct 1-2 sentence replies
- Each suggestion shown in both English and your native language

✍️ **Custom AI Reply**

- Describe what you want to say in your native language
- AI generates a polished, professional English reply
- Example: "Tell the customer we can offer a 10% discount if they order this week"

🎨 **Polished UI**

- WhatsApp mobile-style side panel design
- Dark / Light theme toggle
- Contact search & group filtering
- Auto-resizing input field

🔒 **Privacy & Security**

- All data stored locally (chrome.storage.local)
- No chat logs uploaded to any third-party server
- AI calls use only your own API key, sent directly to your chosen AI provider
- Open source, fully auditable code

**📋 Quick Start**

1. Install the extension and open WhatsApp Web
2. Click the extension icon → "Open Side Panel"
3. Configure your AI API key in the settings popup
4. Start using AI translation and reply suggestions

**🆓 Completely Free**
This is the free edition with all core features included. No license key required, no subscription.

**🔧 Supported AI Providers**

- DeepSeek (recommended, best value)
- OpenAI (GPT-4o / GPT-4)
- OpenRouter (access to various models)
