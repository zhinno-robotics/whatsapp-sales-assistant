隐私政策

最后更新日期：2026 年 6 月 2 日

本隐私政策适用于"WhatsApp Sales Assistant"浏览器扩展（以下简称"本扩展"）。

数据收集与存储

本扩展不收集、不上传、不分享您的任何个人数据。

  • 聊天记录：您的 WhatsApp 聊天消息仅存储在您的浏览器本地（Chrome 的 chrome.storage.local API），不会上传到任何服务器。
  • API 密钥：您配置的 AI 服务商 API Key 存储在浏览器本地，仅用于直接向您选择的 AI 服务商（如 DeepSeek、OpenAI、OpenRouter）发起请求。本扩展不中转、不记录您的 API Key。
  • 设置偏好：语言选择、主题偏好等设置仅存储在本地。

AI 服务调用

当您使用翻译、回复建议等功能时，本扩展会将当前聊天消息的文本内容直接发送给您配置的 AI 服务商（DeepSeek、OpenAI 或 OpenRouter）以生成翻译或回复建议。

  • 消息文本仅用于当次 AI 请求，本扩展不会在您的设备之外保存这些请求内容。
  • AI 请求通过 HTTPS 加密传输。
  • 您发送给 AI 服务商的数据受该服务商的隐私政策约束。请参阅：
    • DeepSeek: https://platform.deepseek.com/privacy
    • OpenAI: https://openai.com/policies/privacy-policy
    • OpenRouter: https://openrouter.ai/privacy

语音消息

免费版本不支持语音转文字功能。本扩展不会处理或上传您的 WhatsApp 语音消息。

数据删除

您可以随时在扩展设置中清除数据，或直接卸载本扩展。卸载后，所有本地存储的数据将被完全删除。

第三方服务

本扩展不包含任何分析 SDK、错误追踪、广告或第三方统计工具。

开源

本扩展为开源软件，源代码可在 GitHub 查看。您可以自行审查代码以验证本隐私政策中的所有声明。

联系我们

如果您对本隐私政策有任何疑问，请通过 GitHub Issues 联系我们。

Privacy Policy

Last updated: June 2, 2026

This Privacy Policy applies to the "WhatsApp Sales Assistant" browser extension ("the Extension").

Data Collection & Storage

The Extension does not collect, upload, or share any of your personal data.

  • Chat Messages: Your WhatsApp chat messages are stored only locally in your browser (using Chrome's chrome.storage.local API) and are never uploaded to any server.
  • API Keys: Your AI provider API keys are stored locally and used solely to make direct requests to your chosen AI provider (DeepSeek, OpenAI, or OpenRouter). The Extension does not proxy or log your API keys.
  • Preferences: Language settings, theme preferences, and other configuration are stored only locally.

AI Service Calls

When you use features like translation or reply suggestions, the Extension sends the text content of the current chat messages directly to your configured AI provider to generate results.

  • Message text is used only for the current AI request and is not persisted beyond your device by the Extension itself.
  • AI requests are transmitted over HTTPS.
  • Data sent to AI providers is subject to their respective privacy policies:
    • DeepSeek: https://platform.deepseek.com/privacy
    • OpenAI: https://openai.com/policies/privacy-policy
    • OpenRouter: https://openrouter.ai/privacy

Voice Messages

The free edition does not include voice-to-text functionality. The Extension does not process or upload your WhatsApp voice messages.

Data Removal

You can clear data at any time through the extension settings, or simply uninstall the Extension. Upon uninstallation, all locally stored data is permanently deleted.

Third-Party Services

The Extension contains no analytics SDKs, error tracking, advertisements, or third-party statistics tools.

Open Source

The Extension is open source. Source code is available on GitHub. You may audit the code to verify all claims in this policy.

Contact

If you have any questions about this Privacy Policy, please contact us via GitHub Issues.
